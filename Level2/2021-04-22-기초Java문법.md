## Enum
- Enum이 뭔가요?
    - Enum은 열거형으로써, 서로 연관된 상수들의 집합이라고 할 수 있습니다. 
    - Java에서의 Enum은 여기에 로직을 추가하여, 서로 연관된 상수 Singleton 객체의 집합이라고 할 수 있습니다. 
- Enum을 왜 사용하죠? 장단점은요?
    - Enum을 사용하는 이유는, 장점이 많기 때문입니다. 
        - 우선 상태와 행위를 동시에 관리하는 코드 작성이 가능해집니다.
